Class default

Represents a user. Most properties are promises, to lazily load them from the server if they're not present.

property

{number} id The id of the user

property

{Promise} name The name of the user

property

{Promise} about The about section of their chat profile

property

{Promise} isModerator True if the user is a moderator, false otherwise

property

{Promise} messageCount The number of all time messages this user has sent

property

{Promise} roomCount All time number of rooms this user has been a part of

property

{Promise} lastSeen The number of seconds since this user was last seen

property

{Promise} lastMessage The number of seconds since this user posted a message in any chat
